# `Coco Battle Royale II`<br/>`ココバトル (kokobatoru) 2`

> homebrew game for nintendo years 80 console, distributed as free software.


## How to build

```SHELL
$ git clone --recurse-submodules https://github.com/psywave-games/coco-battle-royale-2
$ make
```
